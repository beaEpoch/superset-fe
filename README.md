# superset-fe
incubator-superset 项目前端

### 功能
支持 dashboard 中单个图表的过滤功能（原生只能使用 filter box 进行全局过滤）

### 使用
```bash
cd assets
```
```bash
cnpm install
```
```bash
npm run dev
```
将编译完成的 dist 目录替换线上 superset 项目的 dist，重启即可（支持该功能的 superset 后端无需任何改动）
